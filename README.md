## How to use

For users who need to crack it manually:

1. Extract the x64 / x86 zip file inside the `DiskGenius_Pro_Uncracked` folder.
2. Extract the two zip files under the `crack` folder. Copy the two configuration files from `DiskGenius_Crack` into the target `diskgenius` folder. Also, copy `DiskGenius_memory.exe` (or `DiskGenius_MoveFiles.exe`) from the `DiskGeniusBypass` folder into the same `diskgenius` folder.
3. Create a copy of `DiskGenius.exe` named `DiskGenius_.exe` in the target `diskgenius` folder.
4. Run the copied `DiskGenius_memory.exe` with administrator privileges. If it runs normally, no output will be displayed.
5. Double-click `DiskGenius.exe`. Registration will then be successful.
